IDS/IPS Simulation - Detailed Overview

Project Overview: Designed and tested IDS/IPS systems with pattern matching techniques like Aho-Corasick, Boyer-Moore, and Regex to detect intrusions.

Development Approach: Configured the detection systems on a dedicated IDS VM, simulated attacker traffic, and monitored alerts and logs for accuracy.

Challenges & Problem Solved: Ensuring precise detection without false positives, optimizing patterns, and log monitoring. Solved real-world problem of automated intrusion detection evaluation.

What I Learned: Pattern matching algorithms, log analysis, IDS/IPS configuration, and real-time monitoring.

Tech Stack: Snort, OSSEC, Linux, Pattern Matching Algorithms, ELK Stack

Deadlock Detection & Resolution Simulation - Detailed Overview

Project Overview: Simulated deadlocks in multithreaded server environments and implemented detection and resolution strategies using Resource Allocation Graphs and Wait-for Graphs.

Development Approach: Independently designed algorithms to detect circular waits and resolve deadlocks via preemption, rollback, and thread termination in a simulated environment.

Challenges & Problem Solved: Detecting and resolving deadlocks dynamically in a multithreaded environment. Solved the problem of resource contention in concurrent systems.

What I Learned: Concurrency control, deadlock detection algorithms, multithreading, problem-solving, and system simulation techniques.

Tech Stack: Java, Multithreading, Resource Allocation Graphs, Wait-for Graphs
