Hybrid-Based Intrusion Detection Research Project
Objective

This project was conducted as part of my Honours research. The goal was to design and evaluate a hybrid-based Intrusion Detection System (IDS) by comparing the effectiveness of different pattern-matching algorithms in detecting network attacks. The research focused on improving detection accuracy and performance in real-time network environments.

Development & Methodology

The project involved setting up a simulated enterprise network environment using multiple virtual machines:

  • An IDS VM configured with Snort and OSSEC for signature-based detection.
  • An attacker VM used to generate malicious traffic such as ARP spoofing and network scanning.
  • An ELK stack (Elasticsearch, Logstash, Kibana) for log collection and visualization.
  • Evaluation of different pattern-matching techniques including:
    • Aho-Corasick Algorithm
    • Boyer-Moore Algorithm
    • Regular Expressions

The system was tested using various attack simulations to determine which algorithm performed best in terms of speed, resource usage, and detection accuracy.

Challenges
  • Configuring multiple virtual machines to communicate in a controlled environment.
  • Generating realistic attack traffic for accurate evaluation.
  • Comparing algorithm efficiency using measurable performance metrics.
  • Interpreting IDS logs and fine-tuning detection rules.
Learnings
  • Deep understanding of intrusion detection systems and cybersecurity concepts.
  • Practical experience with Snort, OSSEC, and network security monitoring tools.
  • Knowledge of pattern-matching algorithms and their real-world applications.
  • Research methodology, data analysis, and technical report writing.
Tech Stack & Tools

Snort, OSSEC, ELK Stack, Kali Linux, VirtualBox, Python scripting, Pattern Matching Algorithms (Aho-Corasick, Boyer-Moore, Regex), Networking Protocols
